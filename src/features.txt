This file contains the features of the chess engine. and some of the goals

I guess this is a temporary file but maybe not. 

Features:
- bitboard sliding piece lookup
- minimax
- game tree
- battle
- draw by repition
- alpha beta
- negamax
- iterative deepening
- endgame weights
- endgame playstyle 
- piece square tables
- attack square score
- doubled pawn penalty
- passed pawn score
- accompanied pawn bias

Goals:
- transposition table
- quiscience search
- move priority search
- pawn promotion
- king safety bias
- castling bias
